Detailed Description

RViz plugin to display NavMap messages as triangle meshes.

This class inherits from rviz_common::MessageFilterDisplay to subscribe to and process navmap_ros_interfaces::msg::NavMap messages. It renders the geometry in the RViz scene using Ogre primitives, with optional coloring by selected layers and visualization of per-triangle normals.

Note
Typical usage: add the plugin in RViz, choose the "NavMap" display, and select the desired layer to visualize.
